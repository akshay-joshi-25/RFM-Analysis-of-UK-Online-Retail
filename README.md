# 📊 **RFM Analysis of U.K. based Online Retail Firm**

The objective of this project is to perform RFM (Recency, Frequency, Monetary) analysis on a U.K. based online retail company’s transactional data to segment customers based on their purchasing behavior, identify high-value and at-risk customers, and generate actionable insights to support targeted marketing strategies and improve customer retention.
